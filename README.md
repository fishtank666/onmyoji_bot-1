# 阴阳师辅助脚本
图片识别后进行点击，不会鬼使黑

## 功能说明
目前支持的功能：
#### 御魂
* 单人御魂乘客
* 单人御魂司机
* 双开御魂
> 以上功能支持标记己方式神。列如队伍中有大舅妈

#### 探索
* 单人打手
* 单人司机
* 双开
> 自动识别经验怪（准确率90%）、自动挑战boss、自动领取boss奖励

#### 突破
* 自动战斗 （标记式神，适合三拉队伍，大蛇等）
* 手动战斗 （配置式神行动方案，列如茶几需要切被动. 需要代码编写才能实现）
> 自动挑战，失败后重试一次，连续失败，则认为打不过，手动刷新。奖励次数取最大奖励次数挑战。目前没有挑战券，需手动结束脚本

系统自带了一套 茶几切不动的代码。如果有需要可自行编写
